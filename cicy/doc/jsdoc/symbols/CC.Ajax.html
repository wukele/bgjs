类 CC.Ajax

Ajax请求封装类
定义在: cicylib-all-debug.js.

属性 名称说明
 
CC.Ajax()
属性
属性 属性说明
 
asynchronous {Boolean} ,默认值:true
是否异步,默认true
 
busy {Boolean}
指明当前Ajax是否处理请求处理状态,在open后直至close前该值为真.
 
caller {Function}
用于调用failure,success函数的this对象.
 
contentType {String}
默认application/x-www-form-urlencoded
 
data {Object}
send发送的数据
 
disabledComp {DOMElement}
在请求过程中禁止的元素
 
displayPanel {DOMElement|String}
如果数据已加载,数据显示的DOM面板.
 
eventable=true=true {Boolean}
该类是可以监听,发送和处理事件的.
 
failure {Function}
失败后的回调.
 
json {Object}
调用#getJson方法后保存的json对象,在#close方法调用后销毁.
 
method {String}
GET 或者 POST,默认GET
 
msg {String}
提示消息
 
noCache {Boolean} ,默认值:true
是否忽略浏览器缓存,默认为true.
 
onfinal {Function}
无论请求成功与否最终都被调用.
 
success {Function}
设置成功后的回调,默认为运行服务器返回的数据内容.
 
text {String}
调用{@link # getText}方法后保存的text文本,在#close方法调用后销毁
 
timeout {Number} ,默认值:10000
设置超时,默认10000ms
 
url {String}
请求URL
 
xmlDoc {XMLDocument}
调用#getXmlDoc方法后保存的XMLDocument对象,在#close方法调用后销毁.
 
xmlReq {XMLHttpRequest}
对象

Event Attributes Event Name and Description

数据请求失败返回后发送.
 
final(ajax)
请求结束后调用,无论成功与否.

在获得XMLHttpRequest数据调后Ajax.getJson方法后发送,可直接对当前json对象作更改,这样可对返回的json数据作预处理.

在获得XMLHttpRequest数据调后Ajax.getText方法后发送,如果要改变当前text数据,在更改text后设置当前Ajax对象text属性即可,这样可对返回的文件数据作预处理.
 
xmlDoc(doc, ajax)
在获得XMLHttpRequest数据调后Ajax.getXMLDoc方法后发送,可直接对当前xmlDoc对象作更改,这样可对返回的XMLDoc数据作预处理.

getJson()
获得服务器返回数据格式的JSON对象,该方法先调用getText方法获得文本数据,再将数据转为Javascript对象. 可改变返回的text文本数据以达到修改JSON数据的目的,只要在调用getJson重设ajax.text值即可. 在调用过程中会发送text事件.
Returns:
转换后的Javascript对象,如果数据格式有误,返回undefined
getResponseHeader(key)
获得服务器返回的数据头信息.
Parameters:
key
{string} getText()
提供访问返回数据一致性方法,以文本形式提取返回数据,并将该数据保存在当前Ajax实例的text属性中. 调用该方法中发送text事件,在Ajax关闭后Ajax.text会被清空.
Returns:
{string} text 服务器返回的文件数据
{XMLDocument} getXmlDoc()
获得服务器返回数据的XML Document 格式文档对象,该方法调用了XMLHttpRequest.responseXML.documentElement获得XML文档对象. 在调用过程中会发送xmlDoc事件.
Returns:
{XMLDocument} XML Document 文档对象.
invokeHtml()
