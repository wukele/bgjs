Class CC.util.BrushFactory

Package:CC.util
Defined In:brushfactory.js
Class:BrushFactory
Extends:Object
标题画笔工厂,用于根据类型值输出指定格式的字符串.

自带的画笔有:

  • .xx: 保留两位小数的浮点预留画笔
  • .xx%: 保留两位小数的百分比预留画笔



This class is a singleton and cannot be created directly.
