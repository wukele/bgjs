Class CC.layout.CardLayout

Package:CC.layout
Defined In:cicylib-all.js
Class:CardLayout
Extends:Layout
CardLayout,容器内所有子项宽高与容器一致.
new CC.ui.Win({
     // 使得子项iframepanel布满整个window客户区域.

     layout:'card',
     items : [
       {ctype :'iframe', src:'http://www.g.cn' }

     ]
   });
